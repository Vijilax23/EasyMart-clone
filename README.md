# EasyMart Clone

EasyMart Clone is an Amazon-inspired e-commerce frontend website built using HTML and CSS.

## Features

* Responsive Navigation Bar
* Search Bar with Suggestions
* Product Categories
* Account Page
* Orders Page
* Cart Page
* Product Cards with Images
* Footer Section

## Technologies Used

* HTML5
* CSS3
* Font Awesome

## Project Structure

```
EasyMart-clone
│
├── index.html
├── style.css
├── account.html
├── orders.html
├── cart.html
└── README.md
```

## How to Run

1. Download or clone this repository.
2. Open the project folder in VS Code.
3. Open index.html.
4. Run using Live Server.
